# assets 폴더 안내

이 폴더에 아래 이미지 파일들이 필요합니다:

| 파일명                 | 크기      | 설명                    |
|----------------------|-----------|------------------------|
| icon.png             | 1024×1024 | 앱 아이콘               |
| splash.png           | 1284×2778 | 스플래시 화면 이미지      |
| adaptive-icon.png    | 1024×1024 | Android 어댑티브 아이콘   |
| notification-icon.png| 96×96     | 알림 아이콘 (흰색 단색)   |

## 빠른 생성 방법

Expo 기본 자산 생성기로 placeholder 이미지를 자동 생성할 수 있습니다:

```bash
npx expo install expo-asset
# 또는 아래 명령으로 기본 아이콘 자동 생성
npx create-expo-app --template
```

## 실제 아이콘 디자인

- 배경색: #0A0E1A (다크 네이비)
- 텍스트/아이콘: 📊 또는 차트 모양
- 권장 도구: Canva, Figma, Adobe Express
