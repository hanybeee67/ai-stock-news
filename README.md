# 🚀 급등주 알리미

코스피 · 코스닥 · 미국 주식시장을 **30분마다** 자동으로 살펴서
**5% 이상 급등한 종목**을 `회사명 / 상승률 / 현재가`로 보여주는 초간단 앱입니다.

- **API 키 불필요** — 네이버 증권 공개 API(국내) + 야후 파이낸스(미국) 사용
- 파일 하나(`main.py`)로 동작, 브라우저에서 바로 확인
- 화면은 1분마다 자동 갱신, "지금 새로고침" 버튼으로 즉시 조회 가능

## 실행 방법 (Windows)

`start.bat` 을 더블클릭하면 끝. (패키지 설치 → 서버 실행 → 브라우저 자동 열림)

수동 실행 시:

```
pip install -r requirements.txt
python main.py
```

→ 브라우저에서 http://localhost:8000 이 자동으로 열립니다.

## 설정 바꾸기

`main.py` 맨 위의 상수만 수정하면 됩니다.

```python
SURGE_THRESHOLD = 5.0    # 급등 기준 (%)
CHECK_INTERVAL_MIN = 30  # 점검 주기 (분)
PORT = 8000              # 웹 포트
```

## 구조

```
stock management/
├── main.py           ← 서버 + 30분 주기 수집기 + 웹 화면 (전부 이 파일 하나)
├── requirements.txt  ← 필요 패키지 4개
├── start.bat         ← 윈도우 실행 스크립트
└── README.md
```

## 참고

- 장이 열려 있지 않은 시간에는 직전 거래일 기준 등락률이 표시됩니다.
- 미국 목록은 야후 파이낸스 "Day Gainers" 스크리너 기준(주요 종목 위주)입니다.
